# 🚀 SyntecxHub Full Stack Notes App

A Full Stack Notes Management Application built using the MERN Stack.


# 🌐 Live Website

https://syntecxhub-notes-app-backend.vercel.app


# ✨ Features

✅ User Authentication (Signup/Login)

✅ JWT Authentication

✅ Create Notes

✅ Edit & Update Notes

✅ Delete Notes

✅ Archive / Unarchive Notes

✅ Search Notes

✅ Filter Notes

✅ Responsive Design

✅ Toast Notifications

✅ Protected Routes

✅ MongoDB Database Integration

✅ Full Stack Deployment



# 🛠️ Tech Stack

## Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM
- React Hot Toast

## Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication



# 📁 Project Structure

controllers/

routes/
middleware/
frontend/

package.json


# ⚙️ Installation

## Backend Setup

npm install

npm run dev

## Frontend Setup

cd frontend

npm install

npm run dev



# 🔑 Environment Variables

Create `.env` file:

MONGO_URI=your_mongodb_url

JWT_SECRET=your_secret_key



# 🚀 Deployment

Frontend deployed on Vercel

Backend deployed on Render

Database hosted on MongoDB Atlas
